# JEP – Macrocasos

Repositorio con el código y resultados del análisis de macrocasos de la JEP. 

1. El código se presenta por subcasos y a nivel general para uno de los macrocasos, en este sentido, para observar los resultados a un caso correspondiente se debe seleccionar cada uno dependiendo de este criterio
2. Para reproducir el código correspondiente a cada uno de los macrocasos se deberá emplear la carpeta "Casos_JEP", la cual contiene los diccionarios y elementos correspondientes a la depuración de las transcripciones.
